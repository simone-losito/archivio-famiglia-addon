# Archivio Famiglia - Documentazione

Dopo l'installazione apri la Web UI dell'add-on.

## Storage
I documenti vengono salvati nella share Home Assistant:

`/share/archivio`

## Database
Il progetto usa MariaDB/MySQL. Configura i dati DB dalle opzioni add-on.

## Prima pubblicazione
Copia il contenuto attuale del tuo software PHP dentro:

`archivio_famiglia/rootfs/var/www/html/`

prima di pubblicare la repository.
